## Typewriter Effects

Libraries that simulate the visual effect of text being typed out on a screen.

### JavaScript / TypeScript
* [TypeIt](https://typeitjs.com/) - The most versatile animated typing utility on the planet.
* [Typed.js](https://github.com/mattboldt/typed.js) - A JavaScript typing animation library.
* [iType.js](https://github.com/a-lucas/iType.js) - A vanilla JavaScript typing animation library without dependencies.
* [TheaterJS](https://github.com/GabinAureche/TheaterJS) - A typing effect mimicking human behavior.
* [t-writer.js](https://github.com/ChrisCavs/t-writer.js) - A native text typewriter effect without dependencies.

### React
* [react-type-animation](https://github.com/maxeth/react-type-animation) - A React component to animate typing text.
* [react-typing-effect](https://github.com/iamhosseindhv/react-typing-effect) - A UI component for React to type text.
* [typewriter-effect](https://github.com/tameemafi/typewriter-effect) - A simple react component for adding a cool typewriter effect to any project.
* [react-typical](https://github.com/catalinmiron/react-typical) - React typing animation in ~400 bytes of JavaScript.

### Vue
* [vue-typewriter](https://github.com/drewjbartlett/vue-typewriter) - A Vue.js component for typing text.
* [vue-typed-js](https://github.com/Orlandster/vue-typed-js) - A Vue.js wrapper for Typed.js.
* [vue-typer](https://github.com/cngu/vue-typer) - Vue component that simulates a user typing, selecting, and erasing text.

### Angular
* [ngx-typed-js](https://github.com/Orlandster/ngx-typed-js) - An Angular wrapper for Typed.js.

### CSS
* [CSS Typewriter Effect](https://css-tricks.com/snippets/css/typewriter-effect/) - CSS Tricks guide to creating typewriter effects purely with CSS animations.
* [Typewriter CSS](https://github.com/LeaVerou/typewriter) - A neat trick to create a typewriter effect with 100% CSS by Lea Verou.

### Python
* [Rich](https://github.com/Textualize/rich) - Rich text and beautiful formatting in the terminal.
* [tqdm](https://github.com/tqdm/tqdm) - While mainly for progress bars, can be paired with simple `time.sleep()` loops for terminal typewriter text.

### Go
* [pterm](https://github.com/pterm/pterm) - A modern Go module to beautify console output, which includes a built-in interactive TextPrinter for typewriter effects.

### Mobile (Flutter / iOS / Android)
* [animated_text_kit (Flutter)](https://pub.dev/packages/animated_text_kit) - A flutter package which contains a collection of cool text animations, including Typewriter.
* [GhostTypewriter (iOS)](https://github.com/wibosco/GhostTypewriter) - A UIView subclass that adds a typing animation effect to your text.
* [HTextView (Android)](https://github.com/hanks-zyh/HTextView) - An Android text view library with many text animation effects.

## Graphics Programming & Generative Typography

A niche collection of repositories tailored to manipulating fonts, text geometries, and SVGs on a mathematical or procedural level, often used in WebGL, Shaders, or creative coding.

### Text Rendering Engines & Font Parsing
* [HarfBuzz](https://github.com/harfbuzz/harfbuzz) - A text shaping engine that primarily converts a string of Unicode characters into properly positioned glyphs. (Low-level C/C++ engine).
* [FreeType](https://freetype.org/) - A freely available software library to render fonts (low-level, rasterization engine).
* [opentype.js](https://github.com/opentypejs/opentype.js) - A JavaScript parser and writer for TrueType and OpenType fonts. It gives you access to the letterforms of text from the browser or Node.js.
* [fontkit](https://github.com/foliojs/fontkit) - An advanced font engine for Node and the browser, supporting multiple font formats, ligature substitution, and glyph path extraction.

### Creative Coding & WebGL Typography
* [msdfgen](https://github.com/Chlumsky/msdfgen) - Multi-channel signed distance field generator. Essential for rendering crisp, infinitely scalable text in WebGL/OpenGL.
* [troika-three-text](https://github.com/protectwise/troika/tree/master/packages/troika-three-text) - Extremely high-quality text rendering for Three.js using MSDF (Multi-channel Signed Distance Fields) and shader-based rendering.
* [plumin.js](https://github.com/byte-foundry/plumin.js) - Create and manipulate fonts using javascript. Perfect for generative type design.
* [text-to-svg](https://github.com/shrhdk/text-to-svg) - Convert text into SVG paths. Useful for plotting, laser cutting, or generating 3D meshes from text.

### Procedural SVG & Vector Scripting
* [paper.js](https://github.com/paperjs/paper.js) - The Swiss Army Knife of Vector Graphics Scripting. A canvas/SVG based library capable of incredibly complex curve math and generative path typography.
* [Two.js](https://github.com/jonobr1/two.js) - A two-dimensional drawing API geared towards modern web browsers. It renders to SVG, Canvas, and WebGL interchangeably.
* [Vivus](https://github.com/maxwellito/vivus) - A lightweight JavaScript class that allows you to animate SVGs, giving them the appearance of being drawn on the screen (often used for animated SVG typography/icons).
* [Rough.js](https://github.com/rough-stuff/rough) - A small graphics library that lets you draw in a sketchy, hand-drawn-like style. Can be applied to SVG paths and text primitives.
* [Zdog](https://github.com/metafizzy/zdog) - A flat, round, designer-friendly pseudo-3D engine for canvas & SVG. Great for niche 3D rotating icons or text structures.
* [svg.js](https://github.com/svgdotjs/svg.js) - A lightweight library for manipulating and animating SVGs.
